5. Utvärdering

Det som gick bra var att bygga ramen, alltså t ex header, nav, footer osv alltså. Det svåra var sedan att bygga de unika för varje sida. Alltså articles frames. Det som var svårt var att på t ex index sidan anpassa bilden till griddet så att det inte gick ut över boxen och ut på footern. Jag fick faktiskt flera problem som är dokumenterat ovan. Det slutade med att jag gjorde om och gjorde rätt, problemet låg i att jag använde margin istället för padding på fel ställe.

Styrkor - Jag skulle påstå att jag är bra med flex, css grid, html och att bygga ihop sidan.

Svagheter - Jag är dålig på photoshop, att anpassa bilder till sidan.

Jag tror sidan hade blivit mycket bättre om jag la ner tiden i att lära mig att anpassa bilder till css grid osv, för att kunna göra en sida med mer bilder. Sidan hade blivit mer levande med fler bilder. Tror även det kunde blivit en bättre sida om jag hade lagt mer tid på att lära mig mer css istället för att bara använda det jag kunde. Då jag hade mycket jag ville ha men snabbt gav upp på för att det blev för komplicerat.
